# Blackjack Game

A simple Blackjack game built in Python as a practice project.

## Features

- Draw cards (Hit)
- Stop drawing (Stand)
- Random card generation
- Basic win/lose/draw logic
- Replay option

## How it works

1. The player starts with 2 cards
2. The computer shows one card
3. The player can:
   - Draw another card
   - Stop
4. If the player goes over 21 → loses
5. The computer draws and the scores are compared
6. The result is displayed

## Run the project

```bash
python main.py
```
